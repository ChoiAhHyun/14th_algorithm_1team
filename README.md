# 14th_algorithm_1team

1. 각자 계정에 fork

2. 계정에 fork 된 저장소를 git clone 하여 작업공간 생성

3. 생성 된 폴더에서 각자 학습한 내용 및 코드 업로드

4. fork 된 저장소에 push 후, fork 된 원본 저장소에 pull request


-------------------------------------------------------------------------------------------

공통

각자 정해진 주제를 학습 및 정리<br>
그 주차의 주제에 해당되는 프로그래머스에 있는 문제 풀이<br>
정리한 내용 및 코드 각자 깃 폴더에 업로드<br>


벌금 정책<br>
지각 - 2,000 (30분 이상 지각은 결석처리)<br>
결석 - 5,000<br>
준비X - 4,000<br>


스터디 불참 및 지각은 적어도 3일전에 알려주고, 다른 사람들이 인정을 해야 인정 됨



1주차<br>
선택 / 버블 / 삽입 (기수, 계수는 선택사항)


2주차<br>
병합 / 퀵 / 힙 정렬


3주차<br>
이진 / 레드블랙 / B-트리


4주차<br>
다차원 트리 / 해시 테이블


5주차<br>
BFS / DFS

6주차 이후<br>
카카오, 네이버 등 코딩 테스트 문제 풀이
